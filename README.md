# NLP + ML

## Movie Review
Neste notebook do Colab, trabalhei com um dataset do Kaggle contendo avaliações de filmes e seus respectivos sentimentos. Desenvolvi um modelo de machine learning capaz de classificar o sentimento do texto como positivo ou negativo. Para isso, utilizei a biblioteca scikit-learn, a técnica de Bag of Words para transformar o texto em representação numérica e treinei três modelos diferentes para comparar seus desempenhos.

## Classificação de documentos
Utilizei outro dataset do Kaggle composto por textos e suas categorias. O objetivo foi treinar um modelo capaz de classificar automaticamente os documentos com base no conteúdo textual. Neste projeto, utilizei de stop-words para priorizar palavra importantes e FastText para a vetorização dos textos.
